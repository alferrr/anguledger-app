# Anguledger

**Anguledger** is a web-based expense tracker built with **Angular**. It helps users manage their personal finances by recording, categorizing, and visualizing expenses through a clean and interactive dashboard. This project also served as a hands-on learning experience for building applications with the **Angular framework**, focusing on component-based architecture, data binding, and responsive design.

## Key Features

- Record and categorize expenses
- View recent transactions in a table-style layout
- Visualize spending patterns with charts
- Quick-add functionality for fast expense entry
- Responsive and intuitive dashboard design

## Tech Stack

- **Frontend:** Angular, TypeScript, HTML, SCSS
- **Data Visualization:** Chart.js
